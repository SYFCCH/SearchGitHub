# GitHub开源项目搜索
* GitHub常用的搜索条件小结    

1. 搜索名字  in:name xxx
2. 搜索描述 in:description xxx
3. 搜索readme in:readme xxx
4. 按stars stars:>2000
5. 按fork fork:>3000
6. 仓库大小搜索: size:>=5000        5000是5000k
7. 按更新时间 pushed:>2020-01-01
8. 按语言 language:xxx
9. 按作者名 user:xxx [注意:引号后面不要有空格]
10. 搜索的方式可以组合     
11. 更多高级搜索 https://github.com/search/advanced
